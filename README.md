# 🤖 AI Resume & Portfolio Builder

**Build professional resumes and portfolio websites in minutes using AI.**

## ✨ Features

- 🤖 AI Content Generation — plain English to professional resume
- 📋 6 Professional Templates (Modern, Classic, Minimal, Creative, Executive, Developer)
- 📥 PDF Export via Puppeteer
- 🌐 Auto-generated Portfolio Website
- ⚡ AI Regeneration for any section
- 💾 Auto-Save with localStorage
- 📱 Fully Responsive

## 🚀 Quick Start

```bash
git clone https://github.com/istiaqhaider8/ai-resume-portfolio-builder.git
cd ai-resume-portfolio-builder && npm install
cp .env.example .env && npm run dev
```

## 🛠️ Tech Stack

Frontend: HTML5, CSS3, Vanilla JS | Backend: Node.js, Express v5 | AI: OpenAI + local fallback | PDF: Puppeteer
